Minimum HTML/CSS/Javascript Template

This is the minimum set of files for a completely static stand-alone bare bones HTML5 website. This code should work on any mainstream browser from 2018; if you find a bug let me know at @daveseah on Twitter.

There are some features I'm using that may be unfamiliar to casual developers; I've included the year these features became safe to use in mainstream browsers:

  • [1999] Hyperlinks are forced to open in a new window with the <base> tag.
  • [2012] For consistency across all browsers, the base CSS is normalized. The 'normalize.css' stylesheet is included by 'style.css'.
  • [2018] Javascript is loaded as an ECMAscript module, which loads after index.html has completely loaded.

File Locations

  • Home page is 'index.html'
  • CSS loaded from 'css/style.css'
  • Javascript loaded from 'js/main.js'
